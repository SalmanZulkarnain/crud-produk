## Cara Memakai
1. Input nama, harga, dan deskripsi dari produk kalian dibagian ``` Input Form ```
2. Klik button ``` Submit ```
3. Setelah itu, akan muncul ``` Nama Produk ```, ``` Harga ```, dan ``` Deskripsi ```. Kita juga bisa melakukan aksi ``` Edit ``` dan ``` Delete ```
